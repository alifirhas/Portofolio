Meal Planner on Steroid

Meal planner dengan algoritma genetika

MPS dirancang khusus untuk memberikan rekomendasi rencana diet bagi individu yang mengalami obesitas, dengan menggunakan algoritma genetika yang canggih. Tujuan utama dari sistem ini adalah membantu pengguna dalam menurunkan berat badan, dengan menyediakan rekomendasi rencana diet yang disesuaikan dengan kebutuhan gizi mereka.

Sistem saya buat dengan harapan dapat membantu penderita obesitas dan masyarakat umum secara luas. Sistem memberikan rekomendasi rencana diet yang dibentuk dengan menggunakan algoritma genetika berdasarkan kebutuhan gizi atau syarat diet yang dimiliki oleh pengguna.

Meskipun ada beberapa kendala seperti kurangnya data set makanan Indonesia dan cakupan pasien adalah penderita obesitas normal, tetapi aplikasi ini saya buat dengan segenap kerja keras saya.

Dalam pembuatan MPS, saya menggunakan beberapa teknologi berikut:

  Python
  Django
  Dart
 Flutter
  MySQL
 Postman

Di sini, saya telah mengembangkan sebuah proyek yang terdiri dari Django Server sebagai API dan Flutter App sebagai antarmuka pengguna.

Django REST-API Server Di sini, saya menggunakan Django sebagai backend untuk membuat server REST-API dari Meal Planner on Steroid. Server ini bertugas mengelola transaksi data, menghitung kebutuhan gizi, dan memberikan rekomendasi rencana diet menggunakan Algoritma Genetika.
